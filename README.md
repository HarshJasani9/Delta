# Dayflow - Human Resource Management System 

**Dayflow** is a modern, comprehensive Human Resource Management System (HRMS) built using the **MERN Stack**. It is designed to streamline workforce management by providing distinct, secure portals for **Administrators** and **Employees**.

The system replaces manual spreadsheets with an intuitive interface, allowing organizations to manage attendance, leave requests, payroll, and employee profiles efficiently. It features a fully responsive design with **Dark/Light mode** support.

### Key Features
*  Role-Based Access Control:** Secure login/signup for Admins and Employees.
*  Admin Dashboard:** Manage employees, approve/reject leaves, view attendance stats, and oversee payroll.
*  Employee Dashboard:** Check-in/out for attendance, apply for leaves, view salary slips, and manage profiles.
   Smart UI:** Aesthetic interface with Tailwind CSS and toggleable Dark Mode.
*  Document Management:** Upload and manage employee documents.
